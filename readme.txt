== Description ==
Display the matrix with the years, months and days based on the post archives for quick navigation with various customization options available.

= Features Overview =
* Shortcode to display the archive matrix
* Support for RTL orienation
* Fully responsive styling

== Installation ==
= General Requirements =
* PHP: 7.0 or newer
* WordPress: 5.0 or newer

= PHP Notice =
* The plugin doesn't work with PHP 5.6 or older versions.

= WordPress Notice =
* The plugin doesn't work with WordPress 4.9 or older versions.

= Basic Installation =
* Plugin folder in the WordPress plugins folder should be `gd-date-archives`.
* Upload `gd-date-archives` folder to the `/wp-content/plugins/` directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Plugin doesn't have any settings and it doesn't add any panels to WordPress administration.

== Frequently Asked Questions ==
= Where can I configure the plugin? =
The plugin doesn't have any global settings or configuration panels.
